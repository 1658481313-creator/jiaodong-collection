## 三、迭代方案

### V1 — 结构对标（已完成）

**目标**：严格复刻参考站点的页面结构、色彩体系、字体规格和组件样式

**范围**：
- 完整首页所有区块（Header → Hero → Stats → 精选藏品 → 分类浏览 → 收藏家 → 收藏故事 → Footer）
- 严格使用参考站点色彩：主色 #1e3b7a，辅助 #5b6d84，区段底 rgba(175,215,236,0.16)
- 字体规格完全对标（H1 58px / H2 32px / 正文 17px / 标签 13px）
- 藏品卡片使用替代视觉占位（artifact-visual 彩色区块 + 文字标签）

**交付物**：Ardot 设计稿 1 页，1440px 宽度自适应高度

---

### V1.5 — 设计系统组件库（已完成）

**目标**：建立标准化的 Design Library 和组件契约体系，为前端开发提供可直接复用的代码模板

**范围**：
- 创建 TRAE Design Design Library（胶东收藏，ID: VC3AI487-C81IC）
- 建立完整 Design Token 体系：色彩 scale（navy/artifact-blue/artifact-jade/artifact-bronze/text-primary/surface/border）、排版 token、间距/圆角/阴影 token
- 定义 10 个核心组件的 HTML 预览 + JSON 契约：
  - TopNav / Button / FilterTag / ArtifactCard / CollectorCard / NewsCard / StatCard / Notice / Form / ImagePlaceholder
- 聚合组件样式到 components.css，使用 jd- BEM 命名前缀
- 同步更新 4 份项目文档

**交付物**：
- Design Library：colors_and_type.css, css.json, components.css, components/ (11 JSON), preview/ (11 HTML), specs/
- Design Canvas 项目：胶东私人收藏网站.design（首页 + 组件库页面 + 17 张素材图）
- 更新后的 4 份项目文档

### V2 — 真实素材填充

**目标**：用真实藏品图片、收藏家头像替代占位元素

**改进点**：
- 藏品卡片：artifact-visual 区块替换为真实藏品图片（青花瓷罐、和田玉佩、钱币组、设色画轴）
- 收藏家头像：文字圆形替换为真实头像图片
- Hero art 区块：替换为精选藏品大图组合
- 收藏故事：补充真实故事缩略图

**触发条件**：用户提供素材图或授权使用图库

---

### V3 — 细节打磨与响应式

**目标**：像素级精修 + 响应式断点方案

**改进点**：
- 微调间距对齐（8px 基准网格对齐）
- 补充 hover/交互态设计（按钮、卡片、链接）
- 增加阴影与层次优化
- 响应式断点设计稿：768px / 1024px / 1440px 三档
- 补充细节页设计（藏品详情页、收藏家主页）

**触发条件**：V2 真实素材填充完成
