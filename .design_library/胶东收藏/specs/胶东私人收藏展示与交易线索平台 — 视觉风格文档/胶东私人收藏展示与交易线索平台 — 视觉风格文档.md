# 胶东私人收藏展示与交易线索平台 — 视觉风格文档

> 基于 Ardot 设计文件 `701586159182189`（胶东私人收藏网站V1）的画布实测数据编制。  
> 本文档为项目视觉基调的权威定义，所有后续页面、组件和迭代必须遵循本文档的数值与规则。

---

## 一、设计哲学

### 1.1 核心定位

**信息平台而非艺术画廊**。本平台的视觉基调是「深海军蓝 + 无衬线中文 + 信息密集型布局」，属于理性、克制、条理清晰的信息展示风格，而非感性、浪漫的艺术审美风格。

| 维度 | 本项目风格 | 排斥的风格 |
|------|-----------|-----------|
| 色彩 | 深蓝 `#1e3b7a` 主色，冷调克制 | 红色/暖色/金色/博物馆红 |
| 字体 | Noto Sans SC / 微软雅黑，无衬线 | 宋体/衬线/Cormorant Garamond |
| 布局 | 信息密集型，网格化，数据驱动 | 留白极多，诗意散漫 |
| 语气 | 「以馆藏方式展示」— 客观、理性 | 「走进收藏的世界」— 感性、沉浸 |
| 装饰 | 极简，仅用色彩区分分类 | 花纹边框、古典纹样装饰 |

### 1.2 三条硬规则

1. **色调不变** — `#1e3b7a` 为主色锚点，所有辅助色围绕深海军蓝展开，不做暖色系扩展
2. **纯中文** — 不使用英文标题、双语排版，内容语言统一为简体中文
3. **信息优先** — 每个模块必须提供可阅读的信息量（标题 + 描述 + 标签 + 链接），不做纯视觉装饰区

| 角色 | Hex | RGB 系数 | 用途 |

| 轮播标题遮罩 | `rgba(20,35,58,0.85)` | r:0.08, g:0.14, b:0.23, opacity:0.85 | 图片轮播底部深色条 |
| 轮播图片渐变 | — | 渐变线性：`#1e3b7a`(0%) → `#8CADC7`(60%) → `#eef8fd`(100%) | 图片占位区从深蓝到浅蓝的过渡 |
| 非激活竖条 | `#5b6d84` 30% | r:0.36, g:0.43, b:0.52, opacity:0.3 | 轮播标题条未选中状态指示条 |

### 2.6 色彩使用规则

1. **主色 `#1e3b7a` 仅用于强调** — 按钮、链接、标签、头像底色、竖条指示器、分类标签边框/文字
2. **文字色二选一** — 所有标题用 `#15233a`，所有描述/辅助文字用 `#5b6d84`，不使用中间值
3. **区段底色交替** — 白底 Section 与 `#F2F9FC` 底 Section 交替出现，形成视觉节奏
4. **分类色彩编码** — 每个藏品分类使用其专属占位色，用户可通过颜色快速区分分类
5. **深底浅字例外** — 书画类视觉区使用主色深蓝底（`#1e3b7a`），内部文字须为白色或浅色以保对比度

## 三、字体体系

### 3.1 字体族

```css
--font-family: "Microsoft YaHei", "PingFang SC", "Noto Sans SC", Arial, sans-serif;
```

- **首选**：`Microsoft YaHei`（Windows 系统自带）
- **次选**：`PingFang SC`（macOS 系统自带）
- **Fallback**：`Noto Sans SC`（跨平台 Web 字体，设计稿中使用）
- **最终兜底**：`Arial` → `sans-serif`

**设计稿字体**：画布上所有文字节点均使用 `Noto Sans SC`，weight 样式包括 `Regular` / `Medium` / `SemiBold` / `Bold`。前端开发时替换为微软雅黑 / 苹方。

### 3.3 Weight 分配规则

| Weight | 使用场景 |
|--------|---------|
| **700 Bold** | 所有标题（H1/H2/H3）、按钮文字、标签文字、品牌名、当前页导航、Eyebrow、激活态轮播条目、来源署名 |
| **600 SemiBold** | 统计栏数字 |
| **500 Medium** | 更多链接（「更多藏品 →」「更多收藏家 →」「更多动态 →」「更多故事 →」） |
| **400 Regular** | 正文段落、副标题、导航非激活态、描述文字、非激活态轮播条目、时间戳 |

### 3.4 字体使用规则

1. **标题永远 Bold** — H1/H2/H3/Eyebrow/Pill 文字统一 `font-weight: 700`
2. **正文永远 Regular** — 描述、副标题、时间戳统一 `font-weight: 400`
3. **更多链接永远 Medium** — 所有「更多 X →」统一 `font-weight: 500`，与标题 Bold 和正文 Regular 形成三档对比
4. **不做装饰性字体处理** — 不使用 italic、letter-spacing 调整、text-shadow
5. **标题不截断** — Card H3 使用 `textAutoResize: "HEIGHT"` + `width: "fill_container"`，允许换行

## 七、运动与交互

### 7.1 过渡时长

| 级别 | 时长 | 用途 |
|------|------|------|
| Fast | 150ms ease | 按钮 hover、Pill 切换、导航高亮 |
| Normal | 300ms ease | 卡片 hover 上浮 + 阴影、轮播切换 opacity |
| Slow | 500ms ease | 页面区域渐显（仅用于首屏加载动画） |

### 7.2 交互行为定义

| 元素 | Default | Hover | Active/Focus |
|------|---------|-------|-------------|
| Primary Button | bg `#1e3b7a`, text white | bg `#163066`, translateY(-1px), shadow | outline 2px `#1e3b7a`, offset 2px |
| Ghost Button | bg white, border+text `#1e3b7a` | bg `#1e3b7a`, text white | outline 2px `#1e3b7a` |
| 卡片 | 无阴影 | translateY(-2px), shadow `0 4px 12px rgba(0,0,0,0.08)` | — |
| 分类标签 | bg white, text `#1e3b7a` | bg `#1e3b7a`, text white | — |
| 导航链接 | `#5b6d84` Regular | `#1e3b7a` | Bold + `#1e3b7a`（当前页） |
| 更多链接 | 无下划线 | 下划线 | — |
| 轮播图片 | 显示首帧 | 暂停自动播放 | — |

### 7.3 轮播机制

- **自动播放**：5 秒间隔
- **悬停暂停**：鼠标进入轮播区域暂停，离开恢复
- **指示点点击**：切换到对应帧
- **左右联动**：左侧图片帧与右侧标题帧同步切换
- **过渡效果**：opacity 300ms ease

```
└── MoreLink (14px Medium #1e3b7a)   ← 右侧「更多 X →」

SectionSub (17px Regular #5b6d84)    ← 副标题（标题行下方，8px gap）
```

所有 Section（精选藏品、收藏家推荐、收藏动态、收藏故事轮播标题行）的 TitleRow 结构和样式完全统一。
