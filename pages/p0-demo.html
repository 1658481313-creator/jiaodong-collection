P0-1: 缺少 prefers-reduced-motion 支持 必须修复
问题：全站 12 个页面都没有 prefers-reduced-motion 处理。轮播自动切换、卡片 hover 上浮、滚动动画等在用户系统开启"减少动画"后仍然执行。
修复：全局 CSS 加媒体查询冻结动画 + JS 检测后暂停轮播。

点击下方按钮模拟开启/关闭"减少动画"系统设置：

正常模式 — hover 卡片和圆点都有动画
修复前
@media (prefers-reduced-motion: reduce)
不存在于任何文件中。
轮播 setInterval 无条件运行。
卡片 hover 动画无条件执行。
修复后
theme.css 末尾加全局规则：

@media (prefers-reduced-motion: reduce) {
  * { animation-duration: .01ms !important;
      transition-duration: .01ms !important; }
}

轮播 JS 加检测：
if (matchMedia('(prefers-reduced-motion:reduce)').matches) { /* 不启动 setInterval */ }
P0-2: 缺少 skip-to-content 链接 必须修复
问题：所有页面没有"跳到主要内容"快捷链接。键盘用户每次都要 Tab 过整个导航栏才能到达正文。
修复：在每个页面 <body> 开头加一个隐藏链接，聚焦时显示。

用 Tab 键聚焦下方演示区域，或点击上方"跳到主要内容"链接来查看效果：

跳到主要内容
首页 藏品展示 收藏家 收藏故事 收藏动态 服务帮助 入驻联系

← 你已跳过导航，直接到达主要内容区域

修复前
<body> 直接从 header 开始。
没有 skip 链接。
键盘用户需要 Tab 7 次才能到达正文。
修复后
<body>
  <a href="#main" class="sr-only focus:not-sr-only">
    跳到主要内容</a>
  <header>...</header>
  <main id="main">...</main>

Tab 1 次即可跳过导航。
