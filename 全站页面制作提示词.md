# 胶东私人收藏网站 — 全站页面制作提示词

> 基于设计系统「胶东收藏」（ID: VC3AI487-C81IC）和首页画布（pages/index.html）生成。
> 每个提示词可直接用于 TRAE Design 画布页面制作。
> 所有页面必须引用 Design Library 的 `colors_and_type.css` 作为 Token 基础。

## 一、藏品展示页提示词

```
请为「胶东私人收藏网站」创建「藏品展示」页面，保存为 pages/collection.html。

【设计约束】
- 严格引用 Design Library（胶东收藏，ID: VC3AI487-C81IC）的 colors_and_type.css 作为 Token 基础
- 视觉风格必须与首页（pages/index.html）完全一致：深海军蓝主色 #1e3b7a、无衬线中文、信息密集型布局
- 禁止项：红色系、衬线字体、双语排版、花纹边框、大面积渐变、默认态卡片阴影、圆角>8px、italic/text-shadow
- 所有颜色、字号、间距必须使用 CSS 变量引用，禁止硬编码

【页面结构】
1. Header 导航栏（与首页完全一致）
   - 当前页面高亮「藏品展示」
   - Logo「胶东私人收藏」链接回 index.html
   - 其余导航项：收藏家(collectors.html)、收藏故事(stories.html)、收藏动态(updates.html)、入驻联系(contact.html)、搜索平台(search.html)

2. 面包屑导航
   - 路径：首页 > 藏品展示
   - 字号 13px，颜色 --text-primary-300

3. 页面标题区
   - H1：「藏品展示」32px Bold --text-primary-700
   - 副标题：「浏览胶东地区私人收藏的各类藏品」15px --text-primary-300

4. 筛选工具栏
   - 左侧：8 个分类 FilterTag（全部/瓷器/玉器/书画/铜器/钱币/文房/木器/杂项）
     - 选中态：bg --navy-500, text white
     - 未选中态：bg white, border 1px --border-200, text --text-primary-500
     - 尺寸：height 36px, padding 0 14px, border-radius 6px
   - 右侧：搜索输入框（placeholder「搜索藏品名称」）+ 排序下拉（最新/年代/名称）
   - 输入框样式：border 1px --border-200, radius 6px, height 36px, padding 0 12px

5. 统计条
   - 显示「共 XX 件藏品」和「当前显示 X-X 件」
   - 使用 --surface-200 背景

6. 藏品卡片网格（4列 grid，gap 18px）
   - 每张卡片复用首页 ArtifactCard 结构：
     - 图片区（aspect-ratio 4:3，上方圆角 8px）
     - 卡片体（padding 18px）
       - 分类标签 + 年代标签（13px，--text-primary-300，左右分布）
       - 藏品名称 H3（20px Bold，--text-primary-700，2行截断）
       - 描述文字（15px，--text-primary-400，3行截断）
       - 收藏家署名（13px，--text-primary-300）
     - hover 效果：borderColor 加深 + shadow + translateY(-1px)
   - 卡片链接到藏品详情页 collection-detail.html

7. 分页器
   - 居中，使用数字页码 + 上一页/下一页
   - 当前页：bg --navy-500, text white
   - 非当前页：border 1px --border-200, text --text-primary-500
   - 尺寸：36×36px，radius 6px

8. Footer（与首页完全一致）

【示例数据】
使用首页已有的 4 件藏品数据：
- 青花缠枝莲纹罐（瓷器，明代，王先生）
- 清代和田白玉透雕龙纹佩（玉器，清代，刘女士）
- 清代铜钱组（钱币，清代，赵先生）
- 设色山水画轴（书画，清代，孙先生）
补充 4 件虚拟藏品使网格满 2 行（8 列）：
- 民窑青花碗（瓷器，清代）
- 红山文化玉璧（玉器，新石器时代）
- 康熙通宝（钱币，清代）
- 民国粉彩花卉瓶（瓷器，民国）

【连接器】
- 导航「藏品展示」高亮
- 所有卡片「查看详情」→ collection-detail.html
- 分类标签点击 → 本页筛选（同一页面内滚动到顶部）
- Footer 链接按全站 Footer 连接器规则分配
- Logo → index.html
```

```

   右栏（5列）：
   - 分类标签：FilterTag 样式「瓷器」（bg --surface-200, color --navy-500, 不可点击）
   - 藏品名称 H1：26px Bold --text-primary-700

     - 材质：瓷

   - 分割线（border-top 1px --border-200）

     - bg --surface-200, border-left 3px --navy-500
     - 标题 14px Bold，正文 13px --text-primary-400

   - 收藏家姓名 + 地区
   - 收藏方向
   - 「了解更多」按钮 → collector-detail.html

5. 同分类推荐（横向滚动或 4 列网格）
   - 标题：「同分类推荐」H2 32px
   - 4 张 ArtifactCard（复用首页卡片样式）
   - 显示同分类的其他藏品

6. Footer

【示例数据】
以「青花缠枝莲纹罐」为例填充所有字段。

【连接器】
- 面包屑「藏品展示」→ collection.html
- 面包屑「首页」→ index.html
- 收藏家卡片「了解更多」→ collector-detail.html
- 同分类推荐卡片 → collection-detail.html（其他藏品）
- Footer 链接按全站规则
```
