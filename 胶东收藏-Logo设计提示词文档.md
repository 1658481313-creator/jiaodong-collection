## 一、品牌分析

### 1.1 平台定位

胶东收藏是一个聚焦胶东半岛地区（烟台、威海、青岛等地）私人藏品展示与交易线索的信息平台。

| 维度 | 描述 |
|------|------|
| 核心定位 | 信息平台，而非艺术画廊 |
| 目标用户 | 私人收藏家、藏品爱好者、交易经纪人 |
| 业务范围 | 瓷器、玉器、书画、铜器、钱币、文房、木器、杂项共 8 大分类 |
| 地域特征 | 胶东半岛（山东东部沿海） |

### 1.2 视觉风格基因

| 要素 | 规范 |
|------|------|
| 主色 | 深海军蓝 `#1e3b7a` |
| 主文字色 | 深蓝墨色 `#15233a` |
| 辅助文字色 | 灰蓝 `#5b6d84` |
| 字体风格 | 无衬线中文（Noto Sans SC / 微软雅黑 / 苹方） |
| 设计基调 | 理性、克制、条理清晰、信息密集型 |
| 装饰原则 | 极简，不使用花纹、古典纹样、渐变、阴影（默认态） |

### 1.3 藏品分类色彩编码

| 分类 | 色值 | 色彩联想 |
|------|------|----------|
| 瓷器（青花） | `#eef8fd` 冰蓝 | 青花瓷纹样 |
| 玉器 | `#f5fbfe` 极浅绿蓝 | 玉质通透 |
| 铜器/钱币 | `#eaf4fb` 带灰浅蓝 | 铜锈氧化 |
| 书画 | `#1e3b7a` 深蓝 | 墨色浓重 |

### 1.4 设计禁止项

- 禁止暖色系（红色、金色、橙色）
- 禁止衬线字体（宋体、楷体）
- 禁止古典纹样装饰（回纹、祥云、龙凤）
- 禁止大面积渐变
- 禁止双语排版（纯中文）
- 禁止过于感性/浪漫的艺术化表达

## 三、方案详细说明与提示词

### 方案 A — 青花瓷形标志

**设计理念**：以青花瓷瓶的简约几何轮廓为图形主体，融合书卷/档案盒的抽象形态，象征平台的"实物藏品 + 信息存档"双重属性。图形追求最高程度的简洁，可在小尺寸下清晰辨识。

**设计要素**：
- 图形：青花瓷瓶剪影 + 抽象书卷的融合形态
- 色彩：深海军蓝 `#1e3b7a` 单色
- 字体：无衬线粗体中文"胶东收藏"，置于图标右侧
- 风格：扁平设计，无渐变、无阴影、无装饰

## 四、Logo 应用规范建议

### 4.1 尺寸与安全区域

| 应用场景 | 推荐尺寸 | 最小尺寸 |
|----------|----------|----------|
| 网站导航栏 | 高度 32-40px | 高度 24px |
| 网站 favicon | 32x32px / 16x16px | — |
| 移动端 App 图标 | 192x192px | — |
| 社交媒体头像 | 400x400px | — |
| 名片/印刷品 | 宽度 25-35mm | — |

### 4.2 色彩变体

| 变体 | 图标色 | 背景要求 |
|------|--------|----------|
| 标准版 | `#1e3b7a` 深海军蓝 | 白色或浅色背景 |
| 反白版 | `#ffffff` 白色 | `#1e3b7a` 深海军蓝背景 |
| 单色版 | `#15233a` 深蓝墨色 | 任意背景 |
| 灰度版 | `#5b6d84` 灰蓝 | 任意背景 |

### 4.3 禁止用法

- 不得拉伸或压缩 logo 比例
- 不得在低对比度背景上使用标准版（反白版除外）
- 不得添加阴影、发光、描边等效果
- 不得更换字体或修改文字内容
- 不得将 logo 放在杂乱图案背景上
