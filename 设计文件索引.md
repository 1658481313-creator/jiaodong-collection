## 使用说明

1. 点击访问链接可在 Ardot 编辑器中打开对应设计文件
2. 所有前端开发实现应以本索引中的设计稿为视觉基准
3. 设计稿修改后，同步更新关联的设计文档
4. 新增设计文件时，在本索引中追加登记
5. Design Library 组件的 HTML 预览文件是代码模板，前端开发时直接复制 HTML 结构，搭配 colors_and_type.css + components.css 使用
6. Design Canvas 项目是可视化画布，用于快速预览和迭代页面设计
